1. Fork the repository from: "https://github.com/khoj-ai/khoj"

2. Clone your forked repository to your local machine.

3. Use React Developer Tools to locate the component responsible for the logo.

4. You will find a file named `khojLogo.tsx` containing the logo components.

5. Update the <svg> attributes inside the `KhojLogoType` and `KhojLogo` components as needed.

6. Navigate to `khoj/src/interface/web/public/assets/icons` and replace the icon files with your custom versions.

7. Use VS Code global search to find all occurrences of the original asset filenames and replace them with your own.

8. Search for the keywords "Khoj" and "Khoj AI" throughout the codebase and replace them with your desired name.
